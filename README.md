Вариант №17
Практическое задание.Имитация отжига для решения системы нелинейных уравнений.
Определение - Имитация отжига для решения системы нелинейных уравнений — это метод, который преобразует задачу нахождения решения системы нелинейных уравнений в задачу поиска глобального минимума некоторой функции стоимости (например, суммы квадратов невязок).
Объяснение алгоритма: 
1. Постановка как задача оптимизации
1) Мы минимизируем функцию F(x, y), которая всегда ≥ 0.
2) Глобальный минимум F(x, y) = 0 достигается тогда и только тогда, когда (x, y) — решение системы.
3) Это корректная формулировка: мы сводим решение системы к поиску нуля невязки.
2. Начальное решение
1) Случайные x, y ∈ [-10, 10] — разумный выбор, так как:
2) Из уравнения x + y = 7 следует, что значения не могут быть слишком большими по модулю.
3) Ограничение [-10, 10] покрывает возможные решения (например, x=3, y=4 или x=4, y=3 — оба входят в диапазон).
3. Работа имитации отжига
Алгоритм будет:
1) Генерировать случайные соседние точки (например, x ± δ, y ± δ).
2) Вычислять значение F в новой точке.
3) Принимать новое решение, если оно лучше (F меньше), или с некоторой вероятностью, даже если хуже — чтобы избежать локальных минимумов.
4) По мере "охлаждения" (уменьшения температуры T) вероятность принятия худших решений снижается.
4. Преимущества метода в данном случае:
1) Не требует аналитического решения.
2) Устойчив к плохому начальному приближению.
3) Может находить оба решения (при нескольких запусках).
Подходит для более сложных систем, где аналитическое решение невозможно.
5. Недостатки / ограничения:
1) Не гарантирует точного решения — только приближённое, с заданной точностью.
2) Требует подбора параметров (температура, шаг, число итераций).
3) Медленнее, чем прямые численные методы (например, Ньютон), но надёжнее в плане поиска глобального минимума.
Временная сложность и объяснение: O(K), где K — количество итераций.
Вопрос: Объясните основные операции генетического алгоритма: отбор, кроссовер и мутация.
Генетический алгоритм имитирует процесс естественного отбора для поиска оптимальных решений. Отбор — это выбор наиболее приспособленных особей (решений) из текущего поколения на основе их «функции приспособленности» для участия в размножении. Кроссовер (скрещивание) — обмен частями генетической информации между двумя родителями для создания новых потомков, сочетающих их признаки. Мутация — случайное изменение одного или нескольких генов в хромосоме потомка, что обеспечивает генетическое разнообразие и помогает избежать застревания в локальных оптимумах. Эти три операции повторяются в цикле, постепенно улучшая качество популяции решений.
